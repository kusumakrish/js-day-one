Creation Phase:
Creation Phase allocates the memory for the variables and for the function definations with the name of the functions.intially the values of the variables are declered as undefined.

Execution Phase:
Execution phase pick each line of the code and execute the line by performing the operation defined in the line.

Call Stack:
Call stack is a memory where the code will store before going to execution.it stores the code in which the order of the line enter into the stack and executes as last in first out manner.
